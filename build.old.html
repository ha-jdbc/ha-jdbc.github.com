To build from source, first obtain the source code from either:

  • Source code archive
  • Version control

Build requirements

  • JDK 1.7+
  • Maven 3.0+

Building the JAR

mvn package

The generated jar file will reside in the target directory.

Building the project site/documentation

mvn site

The generated documentation will reside in the target/site directory.
